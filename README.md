# Task-1: Cloud Storage Setup using AWS S3

## Platform
- Amazon Web Services (AWS)
- Amazon S3 (Simple Storage Service)

## Objective
To create and configure cloud storage using AWS S3, upload example files, and configure access permissions.

## Steps Performed
1. Created an AWS account and logged into AWS Management Console.
2. Opened the Amazon S3 service.
3. Created an S3 bucket in the Asia Pacific (Mumbai) region.
4. Uploaded example files into the bucket.
5. Configured access permissions using Bucket Policy.
6. Verified public access using the object URL.

## Access Configuration
- Method: Bucket Policy
- Permission: Public Read (s3:GetObject)

## Deliverables
- S3 bucket created
- Example files uploaded
- Access permissions configured
- Public access verified

## Result
Successfully created and configured cloud storage on AWS S3.
